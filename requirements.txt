# FTV Capital Outreach Tracker — Railway Deployment Guide

## What this gives you
- **Live web app** at a permanent URL (e.g. `ftv-tracker.up.railway.app`)
- **Real Outlook sync** — click "Sync Outlook" to pull fresh emails automatically
- **AI chat** that knows all your companies in real time
- **Auto-updates** whenever you sync

---

## Step 1 — Set up Microsoft Azure app (5 min)

1. Go to https://portal.azure.com → **Azure Active Directory** → **App registrations** → **New registration**
2. Name it "FTV Outreach Tracker", click **Register**
3. Note your **Application (client) ID** and **Directory (tenant) ID**
4. Go to **Certificates & secrets** → **New client secret** → copy the **Value**
5. Go to **API permissions** → **Add a permission** → **Microsoft Graph** → **Application permissions**
   - Add: `Mail.Read`, `User.Read.All`
6. Click **Grant admin consent**

---

## Step 2 — Deploy to Railway (5 min)

1. Go to https://railway.app and sign up (free tier = 500 hrs/month)
2. Click **New Project** → **Deploy from GitHub repo**
   - Or: **New Project** → **Empty project** → drag this folder
3. In Railway dashboard → **Variables**, add:
   ```
   AZURE_CLIENT_ID=<from Step 1>
   AZURE_CLIENT_SECRET=<from Step 1>  
   AZURE_TENANT_ID=<from Step 1>
   USER_EMAIL=mloebel@ftvcapital.com
   ANTHROPIC_API_KEY=<from console.anthropic.com>
   ```
4. Railway auto-deploys. Your URL appears in the dashboard.

---

## Step 3 — Update the HTML frontend

In `ftv_outreach_tracker_v3.html`, find this line:
```javascript
const resp = await fetch('https://api.anthropic.com/v1/messages', {
```

Replace with your Railway URL for chat:
```javascript
const resp = await fetch('https://YOUR-APP.up.railway.app/api/chat', {
```

And update the sync button to call:
```javascript
fetch('https://YOUR-APP.up.railway.app/api/sync', {method:'POST'})
```

---

## Step 4 — Optional: Host the HTML on Railway too

Copy `ftv_outreach_tracker_v3.html` into `ftv-tracker-server/static/index.html`
Railway will serve it at your root URL automatically.

---

## Sync schedule
- **Manual**: click "Sync Outlook" in the app
- **Automatic**: add a Railway cron job — Settings → Cron → `0 8 * * *` (8am daily)

---

## Alternative hosting (even simpler)

**Netlify** (static frontend only, no sync):
1. Go to https://netlify.com → drag `ftv_outreach_tracker_v3.html` onto the deploy zone
2. Get a permanent URL instantly. No backend needed for the chat (uses Anthropic directly).

**The difference**: Netlify = chat works, sync requires opening Claude. Railway = everything is live and auto-syncs.
